# 基于多模态融合的吸烟行为检测系统

*基于 YOLOv8 目标检测与 MediaPipe 手部姿态估计的多模态融合方案，实现视频中吸烟、疑似吸烟、未吸烟三种状态的实时智能判定。*

## 项目介绍

**研究背景**

随着公共场所禁烟政策不断推进，传统人工监控方式效率低、难以全天候覆盖。单一模态的检测方法（仅靠目标检测或仅靠姿态估计）在复杂环境中误报率高、鲁棒性差。

**算法功能**

本系统采用 **YOLOv8** 目标检测模型实时识别视频帧中的香烟与嘴部目标，结合 **MediaPipe** 手部关键点检测算法提取手部姿态信息，综合以下四个维度进行决策级融合判定：

| 维度 | 方法 | 权重 |
|------|------|------|
| 目标存在性 | 检测到香烟目标 | 0.5 分 |
| 目标共存关系 | 同时检测到香烟与嘴部 | 1.0 分 |
| 空间位置关联 | 手部靠近嘴部且香烟靠近嘴部 | 2.5 分 |
| 时序动作持续性 | 滑动窗口内手靠近嘴频次达到阈值 | 1.0 分 |

**实现目标**

实现自动化实时吸烟行为预警，输出 **Smoking（吸烟）**、**Suspicious Smoking（疑似吸烟）**、**No Smoking（未吸烟）** 三种分级判定结果，有效降低单一检测方法的误报率。

- **基础运行环境**：Python 3.10+

- **核心依赖组件**：

|------|------|------|
| PyTorch | 2.6.0 | 深度学习框架 |
| ultralytics | 8.3.83 | YOLOv8 模型推理 |

## 安装依赖

通过以下步骤克隆项目并安装全部依赖：

### 1. 克隆项目

```bash
git clone <项目仓库地址>
cd <项目目录>
```

### 2. 安装依赖

```bash
# 使用 requirements.txt 一键安装（推荐）
pip install -r requirements.txt

# 或逐一手动安装
pip install torch==2.6.0
pip install ultralytics==8.3.83
pip install mediapipe==0.10.9
pip install opencv-python
```

### 3. 准备模型权重

将训练好的 YOLOv8 模型权重文件 `best.pt` 放置于项目根目录下（与 `main.py` 同层）。

## 运行项目

### 基本启动

```bash
python main.py
```

### 修改配置

运行前可编辑 `main.py` 顶部的配置参数：

```python
VIDEO_SOURCE = "1.mp4"       # 视频文件路径，设为 0 则使用默认摄像头
MODEL_PATH = "./best.pt"     # YOLOv8 模型权重路径
CONFIDENCE = 0.75            # 检测置信度阈值
PROCESS_EVERY_N_FRAMES = 3   # 每隔 N 帧检测一次（提升性能）
DETECT_WIDTH = 640           # 检测时缩放的宽度
```

## 使用教程

### 操作说明

1. **准备视频**：将待检测的视频文件放入项目目录，或设置 `VIDEO_SOURCE = 0` 启用摄像头实时采集。
2. **启动程序**：运行 `python main.py`，系统自动加载模型并开始检测。
3. **实时显示**：程序弹出检测窗口，实时展示以下内容：
   - 红色框 — 检测到的香烟目标
   - 蓝色框 — 检测到的嘴部目标
   - 手部指尖关键点及指尖到嘴部的连线与距离
   - 四种判定方法的布尔状态
   - **最终判定结果**（绿色=未吸烟，橙色=疑似吸烟，红色=吸烟）
4. **键盘控制**：
   - `Space`（空格键）— 暂停 / 继续播放
   - `ESC` — 退出程序

### 检测流程

```
视频帧输入 → 帧采样（每3帧检测一次）
         ├── YOLOv8 检测 → 香烟框 + 嘴部框
         ├── MediaPipe 检测 → 手部关键点
         ├── 空间距离计算 → 手-嘴距离、烟-嘴距离
         └── 加权评分融合 → Smoking / Suspicious / No Smoking
```

### 自定义调参

可根据实际场景调整以下阈值（位于 `main.py` 顶部）：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `HAND_MOUTH_DISTANCE` | 90 | 手靠近嘴部的距离阈值（像素） |
| `CIGARETTE_MOUTH_DISTANCE` | 90 | 香烟靠近嘴部的距离阈值（像素） |
| `ACTION_WINDOW` | 20 | 动作分析滑动窗口大小（帧数） |
| `ACTION_THRESHOLD` | 8 | 窗口内手靠近嘴判定阈值（帧数） |
